# EventManagementSystem
Made it for a part of Coding Challenge for Mbition GMBH


This is a simple command-line based Event Management System developed in C#.  
It allows users to create, manage, and organize events efficiently.

## Features

- Create events with name, description, date, and location
- List all events (sorted by date)
- View event details by ID
- Update existing events
- Delete events with confirmation
- Search events by name
- Prevent duplicate events (same name & date)
- Input validation for better user experience
- Case-insensitive command handling

## Technologies Used

- C#
- .NET Console Application

## How to Run

1. Open the project in Visual Studio
2. Build and run the application
3. Use the following commands:
create | list | get <id> | update <id> | delete <id> | search <keyword> | help | exit
